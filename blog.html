Important FAQ

What is the Document Object Model (DOM)?

Answer: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of nodes, where each node corresponds to a part of the document (elements, attributes, or text). The DOM allows developers to interact with and manipulate the content, structure, and style of a webpage dynamically using programming languages like JavaScript.

How do you select an element from the DOM?

Answer: I can select an element from the DOM using methods like:
1. document.getElementById('id')
2. document.getElementsByClassName('class')
3. document.getElementsByTagName('tag')
4. document.querySelector('selector')
5. document.querySelectorAll('selector')

What is event delegation in the context of the DOM, and why is it useful?

Answer: Event delegation is a technique in the DOM where instead of attaching event listeners to individual elements, you attach a single event listener to a parent element. This listener can capture events from child elements that bubble up the event chain. It is useful because it improves performance by reducing the number of event listeners and ensures that dynamically added elements also trigger the event without needing to bind new listeners.

How do you manipulate an element's attributes and styles using the DOM?

Answer: To manipulate an element's attributes and styles using the DOM:
1.Get an attribute: element.getAttribute('attributeName')
2.Set an attribute: element.setAttribute('attributeName', 'value')
3.Remove an attribute: element.removeAttribute('attributeName')
4.Get a style property: element.style.propertyName
5.Set a style property: element.style.propertyName = 'value'
6.Add a class: element.classList.add('className')
7.Remove a class: element.classList.remove('className')
